# Login Defender

**Secure Login System with Brute-Force Protection**  
A Python project demonstrating secure password handling, login attempt limits, and lockout timers. Perfect for beginners moving into real cybersecurity concepts.

---

## **Features**

- ✅ Secure password storage using SHA-256 hashing  
- ✅ User registration with password confirmation  
- ✅ Login system with limited attempts  
- ✅ Automatic lockout timer after too many failed logins  
- ✅ Runs 100% locally — no external servers required  
- ✅ Portfolio-ready, professional code structure
